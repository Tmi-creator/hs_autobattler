Данный проект является попыткой обучения RL-агента для игры в Hearthstone: Battlegrounds.

Для этого написана(in progress) собственная среда, похожая на реальную игру и в дальнейшем будут тестироваться различные методы обучения.

Среда лежит в engine/..., остальное(сейчас нерабочее) лежит в основной папке и в models.

# engine:
- combat - реализована боевая система
- configs - база данных карт, всякие константы и прочее
- effects - эффекты, которые применяют различные карты
- entities - сущности, которые используются, юниты, игрок, карты и прочее
- enums - различные маппинги
- event_system - кор файл для системы ивентов, триггеров и всего что с этим связано
- game.py - Игра, которая соединяет в себе все механики вида Таверна-бой-таверна...
- pool - реализация пула существ и их выбора для таверны
- spells - заклинания и их эффекты
- tavern - реализация таверны с роллами, покупкой существ, заморозкой и прочим

# main:
- run_simulation - тесты всего что только можно
- train - обучение модели на текущей среде
- hs_env - среда для модели(будет обновляться)
- test - проверка нашей модели и ее оценка
